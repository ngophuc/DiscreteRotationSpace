# JMIV_2024_BijectiveDiscretizedRotations

# Finite Rotations on Z2: A Hierarchical Framework for Bijectivity and Injectivity Analysis

## Compilation

cd Sources

mkdir build

cd build

cmake .. -DDGtal_DIR=DGTAL_DIR [-DCMAKE_BUILD_TYPE=Release / Debug]

make -j4

## Execution

./AlgoBottomUp

./AlgoTopDown
